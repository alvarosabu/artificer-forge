Artificer Forge is a game development toolkit built on **Nuxt 4** and **TresJS**, designed for creating 3D adventure games with Vue.

## Why TresJS?

TresJS brings Three.js to Vue with a declarative API:

```vue
<template>
  <TresCanvas>
    <TresPerspectiveCamera :position="[0, 2, 5]" />
    <TresMesh>
      <TresBoxGeometry />
      <TresMeshStandardMaterial color="hotpink" />
    </TresMesh>
  </TresCanvas>
</template>
```

Instead of imperative Three.js code, you compose 3D scenes like Vue components.

## Development Philosophy

**Iterate in playground → stabilize → extract to packages**

1. Prototype new features in `@artificer-forge/playground`
2. Test and refine until the API feels right
3. Extract stable code to shared packages
4. Document approaches in this grimoire

## Package Scope

All packages use the `@artificer-forge/*` scope:

- `@artificer-forge/playground` - Sandbox for experiments
- `@artificer-forge/dnd-game` - D&D adventure game
- `@artificer-forge/components` - Shared 3D components
- `@artificer-forge/grimoire` - Documentation (you're here)

## Architecture

```
game-lab/
├── apps/
│   ├── playground/     # Experiments sandbox
│   ├── dnd-game/       # D&D adventure game
│   └── grimoire/       # This documentation
└── packages/
    └── components/     # Shared 3D components
```
