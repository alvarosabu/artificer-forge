# Animations

The `useCharacterAnimations` composable provides a complete animation system for characters.

## Basic Usage

```ts
import { useCharacterAnimations, AnimationName } from '~/composables/useCharacterAnimations'

const { play, stop, currentAnimName, actions } = useCharacterAnimations(rig)

// Play an animation with crossfade
play(AnimationName.RUNNING_A)

// Stop current animation
stop()
```

## The Composable

```ts
export function useCharacterAnimations(
  rig: MaybeRef<Object3D | undefined>,
  rigSize: RigSize = 'Medium',
)
```

### Parameters

| Param | Type | Description |
|-------|------|-------------|
| `rig` | `MaybeRef<Object3D>` | The character's armature/rig object |
| `rigSize` | `'Medium'` | Rig size variant (for animation path) |

### Returns

| Property | Type | Description |
|----------|------|-------------|
| `actions` | `readonly Record<string, AnimationAction>` | All loaded animation actions |
| `currentAction` | `readonly Ref<AnimationAction \| null>` | Currently playing action |
| `currentAnimName` | `readonly Ref<AnimationNameType>` | Name of current animation |
| `play(name, fadeTime?)` | `Function` | Play animation with crossfade |
| `stop(fadeTime?)` | `Function` | Stop current animation |

## Animation Packs

Animations are loaded from separate GLB files organized by category:

```ts
const ANIM_PACKS = [
  'MovementBasic',     // Walk, run, jump
  'MovementAdvanced',  // Dodge, crouch, sneak
  'General',           // Idle, death, hit, interact
  'CombatMelee',       // 1H, 2H, unarmed attacks
  'CombatRanged',      // Bow, gun, magic
  'Simulation',        // Sit, lie, cheer
  'Tools',             // Chop, dig, fish
  'Special',           // Skeleton-specific
]
```

Files are loaded from:
```
/models/animations/Rig_{size}/Rig_{size}_{pack}.glb
```

## Crossfade Transitions

The `play()` function handles smooth transitions between animations:

```ts
function play(name: AnimationNameType, fadeTime = 0.3) {
  const newAction = actions[name]
  if (!newAction) return

  // Fade out current animation
  if (currentAction.value && currentAction.value !== newAction) {
    currentAction.value.fadeOut(fadeTime)
  }

  // Fade in new animation
  newAction.reset().fadeIn(fadeTime).play()
  currentAction.value = newAction
  currentAnimName.value = name
}
```

### Instant Transitions

For immediate switches (no blend), pass `fadeTime = 0`:

```ts
play(AnimationName.DEATH_A, 0)
```

## Auto-Play Idle

The composable automatically plays `Idle_A` when animations load:

```ts
watch(
  () => Object.keys(actions).length,
  (len) => {
    if (len > 0 && !currentAction.value) {
      play(AnimationName.IDLE_A, 0)
    }
  },
  { immediate: true },
)
```
