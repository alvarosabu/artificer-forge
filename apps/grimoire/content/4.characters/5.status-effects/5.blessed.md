A buff effect that displays an amber sparkles badge. Currently badge-only with no 3D shader overlay or particles.

## Badge

::status-effect-badge{effect-id="blessed"}
::

| Property | Value |
|----------|-------|
| Icon | `i-lucide-sparkles` |
| Color | `text-amber-300` |
| Background | `bg-amber-800` |

## Shader Overlay

No shader overlay is applied for blessed.

## Animation Reaction

No animation reaction is triggered for blessed.

## Particles

No particles are spawned for blessed.
