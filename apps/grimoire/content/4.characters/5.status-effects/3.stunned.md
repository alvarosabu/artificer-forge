A crowd control effect that displays a yellow star badge. Currently badge-only with no 3D shader overlay or particles.

## Badge

::status-effect-badge{effect-id="stunned"}
::

| Property | Value |
|----------|-------|
| Icon | `i-lucide-star` |
| Color | `text-yellow-300` |
| Background | `bg-yellow-800` |

## Shader Overlay

No shader overlay is applied for stunned.

## Animation Reaction

No animation reaction is triggered for stunned.

## Particles

No particles are spawned for stunned.
