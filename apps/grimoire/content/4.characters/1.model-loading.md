# Model Loading

Load 3D character models using `useGLTF` from `@tresjs/cientos`.

## Basic Character Component

```vue
<script setup lang="ts">
import { useGLTF } from '@tresjs/cientos'

const { nodes } = useGLTF('/models/characters/hero.glb')

const rig = computed(() => nodes.value?.Rig_Medium)
</script>

<template>
  <primitive
    v-if="rig"
    :object="rig"
    cast-shadow
    receive-shadow
  />
</template>
```

## How It Works

### useGLTF

`useGLTF` returns reactive refs for the loaded model:

- `nodes` - Named objects from the scene graph
- `materials` - Named materials
- `scene` - The root scene object
- `animations` - Animation clips (if embedded)

### Finding the Rig

Models typically have a named armature/rig object. Access it via `nodes.value?.RigName`:

```ts
// For Synty-style models, rigs are named by size
const rig = computed(() => nodes.value?.Rig_Medium)
```

### Rendering with Primitive

The `<primitive>` component renders any Three.js object:

```vue
<primitive
  v-if="rig"
  :object="rig"
  cast-shadow      <!-- Cast shadows -->
  receive-shadow   <!-- Receive shadows -->
/>
```

::alert{type="info"}
Always guard with `v-if` - `useGLTF` loads asynchronously, so `nodes` is initially empty.
::

## Exposing Controls

To control the character from parent components, expose methods via `defineExpose`:

```vue
<script setup lang="ts">
import { useCharacterAnimations } from '~/composables/useCharacterAnimations'

const { play, stop, currentAnimName, actions } = useCharacterAnimations(rig)

defineExpose({
  play,
  stop,
  currentAnimName,
  actions,
})
</script>
```

Parent can then use template refs:

```vue
<script setup>
const character = ref()

function attack() {
  character.value?.play('Melee_1H_Attack_Chop')
}
</script>

<template>
  <Character ref="character" />
</template>
```

## Model Sources

Recommended model packs for RPG characters:

- [Synty Polygon](https://syntystore.com/) - Low-poly stylized characters
- [Mixamo](https://www.mixamo.com/) - Free rigged characters + animations
- [Ready Player Me](https://readyplayer.me/) - Avatar system
