# D&D Game

D&D-style adventure game built on Artificer Forge patterns.

## Commands

```bash
pnpm -F @artificer-forge/dnd-game dev
pnpm -F @artificer-forge/dnd-game build
```

## Notes

- Uses patterns prototyped in `@artificer-forge/playground`
- Imports shared components from `@artificer-forge/components`
- See playground CLAUDE.md for entity system details
